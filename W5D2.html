    Before starting please check if you have npm installed on your computer by typing in your terminal: 
    npm -v
    If it provides a version number such as 3.8.9: Install nodemon by pasting in your terminal: 
    npm install -g nodemon
    If it says 'npm: command not found' then paste the following three lines in your console: 
    ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
    brew install node 
    npm install -g nodemon
